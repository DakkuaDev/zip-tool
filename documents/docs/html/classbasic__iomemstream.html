Detailed Description

template<typename ELEM_TYPE, typename TRAITS_TYPE>
class basic_iomemstream< ELEM_TYPE, TRAITS_TYPE >

Basic input/output memory stream. Combines imemstream & omemstream. Creates input/output stream around the memory buffer. Supports seeking. Returns EOF when stream seeks behind the size of buffer. Sets badbit if the stream wants to write behind the buffer size.
