
# Zip Tool
Middleware zip tool solution using C++, Lua and Qt framework

# Modules:
- Fronted (Program Interface)
- Backend (Data and Public Methodes)
- Save, Update, Delete and Open Zip Files
- Serialization
- Lua Scripting
